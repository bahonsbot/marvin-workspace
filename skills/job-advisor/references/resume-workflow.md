# Resume Workflow

Use this guide for new resumes, substantial rewrites, and role-specific tailoring.

## 1. Establish the brief

Collect or confirm:
- target role
- company
- location / remote constraints
- seniority level
- job description
- current resume or raw career notes
- any portfolio / LinkedIn / proof links worth using

If the user only gives a vague target like "marketing jobs", narrow it before rewriting.

## 2. Extract the role signal

From the job description, pull out:
- core responsibilities
- must-have skills / tools
- likely ATS keywords
- seniority signals
- proof expectations

Separate true must-haves from nice-to-haves.

## 3. Map evidence before writing

For each major requirement, look for candidate evidence:
- direct experience
- adjacent transferable evidence
- partial evidence
- missing evidence

Do not paper over missing evidence. Call it out.

## 4. Choose structure

Default: reverse-chronological.

Use combination only if the user has a mixed background that still supports the role.
Use functional only when chronology would actively hide the strongest case.

## 5. Rewrite for impact

### Summary
Keep it short: 2-4 lines.

It should answer:
- who the candidate is
- what kind of role they fit
- what evidence makes them credible

Avoid empty claims like:
- hard-working
- passionate professional
- dynamic team player
- results-driven expert

unless immediately backed by real proof.

### Experience bullets
Prefer this pattern:
- action
- scope
- result
- metric if available

Good:
- Built X, improving Y by Z%
- Led A across B teams, reducing C
- Managed D volume / revenue / users

Weak:
- Responsible for...
- Worked on...
- Helped with...

### Skills
Mirror the role language where truthful.
Group skills cleanly.
Do not create bloated keyword dumps.

## 6. Tailor without lying

You may:
- reorder bullets
- rename a section for clarity if still truthful
- foreground relevant projects
- adjust summary and skills language to match the role

You may not:
- invent tools
- invent metrics
- invent leadership scope
- imply expertise where there is only exposure

## 7. Finish with a recruiter check

Before finalizing, check:
- can a recruiter understand the fit in 10-15 seconds?
- are the strongest relevant bullets near the top?
- are there concrete outcomes?
- are dates, titles, and chronology clean?
- does the draft still sound like a real person?

## Output suggestion

When delivering a tailored resume, include:
1. short match read
2. revised resume
3. missing evidence / risk list
4. optional next improvements
