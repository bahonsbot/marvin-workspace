# Research History

Store one file per substantial matchup or market-research session to preserve progression over time.

## Naming convention
`YYYY-MM-DD-<sport-or-event>-research-<NN>.md`

Examples:
- `2026-04-10-premier-league-research-01.md`
- `2026-04-10-lakers-vs-celtics-research-02.md`

## Required fields per research note
- Date/time
- Sport / event / market
- Question being assessed
- Thesis
- Evidence used
- Risks / uncertainty
- Confidence level (capped at 95%)
- Disclaimer / caution note
- Next watch item

## Minimal template
```md
# <date> <sport/event> research <NN>

## Objective

## Thesis

## Evidence

## Risks / uncertainty

## Confidence

## Next watch item
```
